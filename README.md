### 一些英文

* cellpadding:单元格边距

* cellspacing:单元格间距

* border:边界(常用于列表)

* col:即column，列

* align:排成一列(作用为水平对齐)

* span: 跨越(常用于colspan与rowspan，即合并单元格)

* margin: 边缘、边距(常用于设置所有外边距属性)

### 一些英文

* cellpadding:单元格边距

* cellspacing:单元格间距

* border:边界(常用于表格)

* col:即column，列

* align:排成一列(作用为水平对齐)

* span: 跨越(常用于colspan与rowspan，即合并单元格)

* margin: 边缘、边距(常用于设置所有外边距属性)

* caption: 标题、说明(常用于表格标题)

* div:即division,区、区域、部分、界限(div元素是块级元素，它可用于组合其他 HTML 元素的容器)

* option:选择(常用于下拉框选项中)
